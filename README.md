# Zemljevid znanja (POC)

Interaktiven prototip pogleda **Zemljevid znanja** za učno platformo Astra AI. Prikazuje graf tem po posameznem predmetu, obarvan glede na raven znanja, ter stranski pregled napredka.

Live na: https://quattro04.github.io/zemljevid-znanja-poc/

## Značilnosti

- **Graf znanja** — osrednje vozlišče predmeta z orbitirajočimi temami; povezave in oznake so obarvane glede na oceno (zelena ≥ 70 %, rumena ≥ 50 %, rdeča < 50 %).
- **Predmeti** — Matematika, Fizika, Kemija, Biologija in Angleščina; vsak ima svoj graf.
- **Animacije** — postopno pojavljanje vozlišč, animirane napredovalne črtice in obroč skupnega znanja; poudarek ob prehodu miške in izbiri.
- **Stranski panel** — obroč »Skupno znanje«, razčlenitev Močno / Srednje / Šibko in seznam »Priporočeno za ponovitev«.
- **Zoom** — gumba − / + za povečavo grafa (60–160 %).
- **Odziven** — prilagojena postavitev za tablice in telefone.

## Uporaba

Odpri `index.html` v brskalniku. Brez odvisnosti in brez gradnje — vse je v eni datoteki (HTML, CSS in JS).

## Vsebina

- `index.html` — celotna aplikacija.
